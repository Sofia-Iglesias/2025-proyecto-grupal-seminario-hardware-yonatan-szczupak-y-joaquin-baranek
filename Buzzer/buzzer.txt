Nombre del componente: (Buzzer)
¿Qué tipo de dispositivo es (sensor o actuador)? Explicá brevemente su función principal.
El buzzer es un actuador, su función principal es que al estar activado emite un pitido. En nuestro proyecto el buzzer se activará cuando el auto se encuentre con un obstáculo.
¿Qué tipo de señal utiliza para comunicarse con Arduino?
recibe una señal analogica, cuando se perciba un obstaculo con el US el  buzzer se irá activando
¿Cuál es el rango de valores que puede entregar o recibir el componente?
En nuestro caso el buzzer recibirá de 0 a 255 unidades
¿Qué pines de conexión tiene el componente? Ennumerar y describir su función.
Tiene dos pines de conexión. El negativo se conecta al GND vía proto y sirve para cerrar el circuito y permitir que fluya la corriente.
El pin positivo lo conectamos a un pin analogico, este pin positivo tiene la función de recibir la corriente para que se active y emita el pitido el buzzer.
¿Qué pin o pines de Arduino utilizan para conectarlo? ¿Por qué elegiste esos pines?
Utilizamos dos pines del arduino para conectarlo, el GND para cerrar el circuito. También utilizamos el pin 5 del arduino que es analogico para poder controlar cuando el buzzer incrementa su capacidad utilizada.
